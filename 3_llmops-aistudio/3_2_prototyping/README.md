# ラボ 3.2 シナリオ 1: プロンプト フローを使用して第 1 世代 AI アプリケーションのプロトタイプを作成する

## 概要
このラボでは、Prompt フローを使用して生成 AI アプリケーションのプロトタイプを作成する方法を学習します。プロンプト フローを備えた Azure AI Studio は、LLM のアイデアを作成して探索し、特定のユース ケースを説得するのに役立ちます。
